Перевод из 8-й системы счисления в другие системы счисления.

1. Переведите 1238 в 10-ую систему счисления.

  • reorder
    Пример решения:
    Для перевода числа 11000101 в десятичную систему воспользуемся формулой:
    An = an-1 ∙ qn-1 + an-2 ∙ qn-2 + ∙∙∙ + a0 ∙ q0
    Отсюда:
    11011002 = 1·26+1·25+0·24+1·23+1·22+0·21+0·1 = 10810
    Ответ: 108.


2. Переведите число 268 в 16-ую систему счисления.

  • reorder
    Пример решения:
    С помощью последовательного деления на 16 выполняем перевод:
    26/16 = 1, остаток: 10, 10 = A
    1/16 = 0, остаток: 1
    2610 = 1A16
    Ответ: 1А.


3. Переведите число 1238 во 2-ую систему счисления.

  • reorder
    Пример решения:
    Сначала нужно 1238 перевести в 10-ую систему счисления:
    Переводим 1238 в десятичную систему: 1238 = 1·82+2·81+3·1 = 8310
    Далее при помощи последовательного деления на 2 мы переводим 8310 в 2-ую систему счисления.
    83/2 = 41, остаток: 1
    41/2 = 20, остаток: 1
    20/2 = 10, остаток: 0
    10/2 = 5, остаток: 0
    5/2 = 2, остаток: 1
    2/2 = 1, остаток: 0
    1/2 = 0, остаток: 1
    8310 = 10100112
    Ответ: 1010011.
