1. Укажите через запятую в порядке возрастания все основания систем счисления, в которых запись числа 22 оканчивается на 4.

  • reorder
    Пример решения:
    Для начала нужно найти все целые числа N > 4 (т.к. последней цифрой числа должна быть 4) такие, что статок от деления 22 на N равен 4
    Следовательно получаем уравнение: 22=k*N+4, где k - целое неотрицательное числи(0, 1, 2, ...);
    Из формулы получаем, что k*N=18 ,так что задача сводится к тому, чтобы найти все делители числа 18, которые больше 4;
    В этой задаче есть только три таких делителя: 6, 9, 18.

    Ответ: 6, 9, 18.


2. В системе счисления с некоторым основанием десятичное число 129 записывается как 1004. Укажите это основание.

  • reorder
    Пример решения:
    Для начала нужно найти все целые числа N > 4 (т.к. последней цифрой числа 1004 является 4) такие, что статок от деления 129 на N равен 4;
    Обозначим за х нужную систему счисления, тогда будет выполняться тождество(перевод из системы счисления с основанием х в десятичную):
    x3+4*x0=129
    x3+4*1=129
    x3=125
    ∛125 = 5
    Следовательно x = 5

    Ответ: 5.


3. Укажите наименьшее основание системы счисления, в которой запись числа 70 трехзначна.

  • reorder
    Пример решения:
    Обозначим через N неизвестное основание системы счисления, тогда запись числа 70 в этой системе имеет вид:

    x * y * z N = 70,

    Вспомним алгоритм перевода числа из системы счисления с основанием N в десятичную систему: расставляем сверху номера разрядов и умножаем каждую цифру на основание в степени, равной разряду:
    xyzN = x * N2 + y * N + z = 70;
    Поскольку запись трехзначная, x ≠ 0, поэтому 70 ≥ N2;
    С другой стороны, четвертой цифры нет, то есть, в третьем разряде – ноль, поэтому 70 < N3;
    Объединяя последние два условия, получаем, что искомое основание N удовлетворяет двойному неравенству:

    N2 ≤ 70 < N3

    Учитывая, что N – целое число, методом подбора находим целые решения этого неравенства; их несколько – 5, 6, 7, 8:

    52 = 25 ≤ 70 < 53 = 125

    62 = 36 ≤ 70 < 63 = 216

    72 = 49 ≤ 70 < 73 = 343

    82 = 64 ≤ 70 < 83 = 512

    Минимальное из этих значений – 5.

    Ответ: 5.


4. Запись числа 658 в некоторой системе счисления выглядит так: 311N. Найдите основание системы счисления N.

  • reorder
    Пример решения:
    Поскольку 65 < 311 соответственно 8 > N.
    Глядя на разницу между 65 и 311, можно предположить, что N как минимум в 2 раза меньше 8.
    Предположим что N=4. Проверяем:

    3114 = 5310 = 658.

    Таким образом ответ - 4.

    Ответ: 4.


5. Запись числа 344 в некоторой системе счисления выглядит так: 1A8N. Найдите основание системы счисления N.

  • reorder
    Пример решения:
    Для начала нужно найти все целые числа N > 8 (т.к. последней цифрой числа 1A8 является 8) такие, что статок от деления 344 на N равен 8;
    Обозначим за х нужную систему счисления, тогда будет выполняться тождество(перевод из системы счисления с основанием х в десятичную):
    1*x2+A*x1+8*x0=344.
    A16=10(Смотри дополнительные материалы).
    x2+10x+8=344.
    x2+10x-336=0.
    С помощью Теоремы Пифагора получаем дискриминант равный 1444, который равен 382.
    Первый х равен 14, второй -24.
    Осинование системы счисления не может быть отрицательным,
    Таким образом ответ - 14.

    Ответ: 14.


6. Запись числа 180 в системе счисления с основанием N содержит 3 цифры и оканчивается на 0. Перечислите в порядке возрастания все возможные основания системы счисления.

  • reorder
    Пример решения:
    Поскольку запись в системе счисления с основанием N заканчивается на 0, то остаток от деления числа 180 на N равен 0, то есть при некотором целом x имеем N умножить на x = 180.
    Cледовательно, основание N – это делитель числа 180.
    C другой стороны, запись числа содержит 3 цифры, то есть 1000n > 180 ≥ 100n, N3 > 180 ≥ N2.
    Начнем выписывать кубы и квадраты делителей, пока квадрат делителя не будет превышать 180:
    22=4; 32=9; 42=16; 52=25; 62=36; 92=81; 102=100; 122=144;
    23=8; 33=27; 43=64; 53=125; 63=216; 93=729; 103=1000; 123=1728;
    Видим, что из этого списка все условия выполняются для чисел N = 6, 9, 10, 12.

    Ответ:N = 6, 9, 10, 12.
