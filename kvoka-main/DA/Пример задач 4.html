Перевод из 2-й системы счисления в другие системы счисления.

1. Переводите 11011002 в десятичную систему

  • reorder
    Пример решения:
    Для перевода числа 11000101 в десятичную систему воспользуемся формулой:
    An = an-1 ∙ qn-1 + an-2 ∙ qn-2 + ∙∙∙ + a0 ∙ q0
    Отсюда:
    11011002 = 1·26+1·25+0·24+1·23+1·22+0·21+0·1 = 10810
    Ответ: 108.


2. Переведите число 11011002 в 8-ую систему счисления.

  • reorder
    Пример решения:
    Сначала нужно перевести 11011002 в 10-ую систему счисления. Дальше последовательным делением полученное число переводим в 8-ую систему счисления.
    Переводим 11011002 в десятичную систему:
    11011002 = 1·26+1·25+0·24+1·23+1·22+0·21+0·1 = 10810
    Переведем 10810 в 8-ую систему:
    108/8 = 13, остаток: 4
    13/8 = 1, остаток: 5
    1/8 = 0, остаток: 1
    10810 = 1548
    Ответ: 154.


3. Переведите число 11111002 в 16-ую систему счисления.

  • reorder
    Пример решения:
    Сначала нужно перевести 11111002 в 10-ую систему счисления. Дальше последовательным делением полученное число переводим в 16-ую систему счисления.
    Переводим 11111002 в десятичную систему:
    11111002 = 1·26+1·25+1·24+1·23+1·22+0·21+0·1 = 12410
    Переводим целую часть 12410 в 16-ую систему последовательным делением на 16:
    124/16 = 7, остаток: 12, 12 = C
    7/16 = 0, остаток: 7
    12410 = 7C16
    Ответ: 7С.
