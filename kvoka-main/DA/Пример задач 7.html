Перевод из 16-й системы счисления в другие системы счисления.

1. Переведите FB16 в 2-ую систему счисления.

  • reorder
    Пример решения:
    Переведем FB16 в 10-ую систему счисления с помощью формулы
    An = an-1 ∙ qn-1 + an-2 ∙ qn-2 + ∙∙∙ + a0 ∙ q0
    FB16 = 15·161+11·1 = 25110
    При помощи последовательного деления переводим 25110 в 2-ую систему счисления:
    251/2 = 125, остаток: 1
    125/2 = 62, остаток: 1
    62/2 = 31, остаток: 0
    31/2 = 15, остаток: 1
    15/2 = 7, остаток: 1
    7/2 = 3, остаток: 1
    3/2 = 1, остаток: 1
    1/2 = 0, остаток: 1
    25110 = 111110112
    Ответ: 11111011.
